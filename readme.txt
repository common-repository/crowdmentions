Use a command to mention a collection of members on your BuddyPress site.

== Description ==

This BuddyPress plugin sends notifications to a collection of members when you use a command to mention them. Supported commands are:

 - @friends
 - @group
 - @moderators
 - @administrators

== Installation ==

1. Go to Plugins > Add New and search for `crowdmentions`
2. Click on Install Now

You can then go ahead and activate the plugin.

== Frequently Asked Questions ==

= Is the plugin network compatible? =

Yes.

= Do mention commands trigger email notifications? =

No. Mention commands trigger web notifications only.

= Can mention commands be used as usernames? =

Yes but it isn't recommended.

= Can I translate into another language? =

Yes. You can change all output using the "crowdmentions" text domain.

= Where should I submit bug reports? =

If you think you've spotted a bug, please open an issue on [GitHub](https://github.com/henrywright/crowdmentions/issues).
